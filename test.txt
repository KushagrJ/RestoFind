* Node.js is a runtime environment that allows you to run JavaScript code on the
  server (i.e. outside a web browser).
* Express.js is a framework that works on top of Node.js.
  It provides various mechanisms to handle common server-side tasks.
* EJS (Embedded JavaScript) is a templating language that lets you generate HTML
  markup with plain JavaScript.
* Mongoose.js is an Object-Document Mapper library that allows you to interact
  with MongoDB (which is a document database) using JavaScript objects.

* From within a folder, use npm init to create a package.json file for the
  corresponding Node.js project.
* npm init -y will create the package.json file using the default settings.

* Use sudo npm i express mongoose ejs to install express, mongoose and ejs for
  the corresponding Node.js project.
